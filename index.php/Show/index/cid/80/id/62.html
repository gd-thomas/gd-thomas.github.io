THOMAS托玛斯微型气泵5大行业应用

2016-07-22

一、医疗

在选择医疗设备用泵时, 泵的长寿命及低噪音是被考虑的两个关键因素。

Thomas 产品以可靠性及静音运行著称，可用于各种用于监控、消毒、诊断及分析的设备，以及其它应用以帮助病人康复，提高病人舒适度和维持生命。

Thomas 产品典型应用包括：

  • 充气床垫

  • 吸引器

  • 灭菌器

  • 浴缸座椅

  • 血液分析仪

  • 血压监护仪

  • 升降椅

  • 压力治疗仪

  • 移动式牙科治疗车

  • 透析机

  • 药物输送

  • 气体监控

  • 雾化器

  • 制氮机

  • 制氧器

  • 呼吸机

  • 伤口透气

 

二、实验室

在要求苛刻的实验室环境中，技术人员准备对生物和化学样本进行分析的时候，必须依靠设备的可靠性和精度实现多种个关键功能。

Welch 真空泵结构紧凑，极具成本效益，可靠而且不需经常维护，适合需要进行蒸馏、干燥及浓缩操作的实验室应用。

Thomas和Welch产品的典型应用包括：

  • 吸取

  • 吸取是通过吸入清除液体或气体的过程。实验室吸取装置一般包括待吸取的样本（瓶子内、溢出的液体和多孔板上的等）、液体捕集器和真空泵。研究人员一般在离心过滤以收集试验台溢出的液体后，使用吸取装置清除浮在各种深孔板上的液体。

  • 细胞收集

  • 细胞收集是收集动物或人体血液或骨骼骨髓细胞的过程。细胞收集器和真空泵是完成该过程的两个关键设备。

  • 浓缩器

  • 真空浓缩器利用离心力、热量和真空，快速蒸发和浓缩生物样本和分析样本内的溶液。浓缩器和真空泵是完成该过程的两个关键设备。

  • 干燥

  • 干燥器是一个圆形或方形的容器，通常由玻璃、塑胶或不锈钢制成，用于平衡无湿气的受控环境内的物质。干燥器和真空泵是完成该过程的两个关键设备。

  • 过滤

  • 真空过滤利用差压（滤纸上面 的  为大气，滤纸下为真空）驱使溶液通过滤纸。过滤瓶和真空泵是用于完成该过程的两个关键设备。

  • 冷冻干燥

  • 采用冻干法，通过升华除去各种物质内的水份、有机溶剂及/或酸。若要完成该操作，必须使用冻干机及真空泵。

  • 移液器

  • 这是一种手持式器件，重量轻，适合与标准刻度吸液管一起使用，用于精密、安全、不会产生疲劳磨损的产品系列。借助微型真空泵，可执行类似快慢抽吸、调整液面或排空吸液管等工作。

  • 旋转蒸发仪

  • 旋转蒸发器是用在实验室中，对旋转的容器加热并抽真空的状态下使液体快速蒸发。旋转式蒸发器、真空泵、真空控制器及冷却器是完成该操作不可或缺的器件。

  • 溶剂脱气

  • 溶剂脱气是指除去溶剂内的微粒和溶解气体。 很多仪器制造商建议在溶剂和样品过滤及脱气后再行分析。 溶剂内的颗粒及溶解气体可能导致假峰值或基线不稳定。研究人员通过脱气减少仪器停机时间，延长 寿命，优化仪器性能。溶剂必须使用真空泵进行脱气。

  • 组织培养

  • 组织培养就是保持组织活性并让其在培养基上生长的技术或工艺，在过程中，需要用到真空泵将培养基从容器中分离。组织培养抽吸装置一般包括集液器、真空泵及移液器，以便除去培养基。

  • 真空浓缩

  • 真空浓缩器可快速、方便地浓缩临床试验或研究样本，以进行分析。

  • 真空炉

  • 牙科实验室正采用特殊的烤箱来制造陶瓷假牙。为排除软化的陶瓷假牙内的空气或其它气体，烤箱必须被抽空。因此以防止假牙内出现空洞。

 

三、汽车

Thomas 产品面向汽车行业市场，专门针对要求高效率及持续性的应用，如：充气座椅、燃料电池、空气悬挂及轮胎充气。 
Thomas 工程师与主要的汽车制造商及其直接供应商紧密合作，开发极大影响未来驱动器发展的新技术。

Thomas 产品典型应用包括：

  • 气动离合器 
    气动离合器使用少量空气加速 离合器盘的 接合和分离，集盘式离合器所有显著特点与直接空气接合的所有优势于一身。此类离合器用于电动机驱动的负载需急速停止且不再滑移的机械行业。

  • 气动门 
    大多数情况下，气动门都是大客车和租车客车的标准配置。气动门为操作员打开车门此类车辆大车门时提供便利。

  • 气动喇叭 
    气动喇叭形状和尺寸各异，用于船只、半拖车、消防设备、公共汽车和急救车辆行业。

  • 充气座椅 
    充气座椅通过一组内置式气囊系统使座椅变得舒适。该系统包括气泵、控制电子装置、气动阀和开关面板。气泵根据用户需要充气或放气。可通过人的体重将空气挤出，实现被动放气，也可由泵产生的真空进行主动放气。可加入按摩过程。

  • 空气悬挂 
    空气悬挂现在用于汽车和 SUV（运动型多功能车）。一些高端豪华车和卡车将空气悬挂系统作为标准配置。一台压缩机向位于轴或每个车轮下面的缓冲装置或橡胶气囊供气。

  • 制动辅助装置 
    制动辅助装置在紧急情况时帮助减少制动距离和加强保护。电子控制系统监控制动器速度，识别"紧急停车"，在几秒内启动完全制动，将汽车制动距离减少多达 20%。如今多种车辆正采用该技术，包括 SUV、卡车和私家车。

  • 燃料电池 
    燃料电池是一种化学能转化电能装置，原理是氢与氧反应生成水，在反应过程中产生电和热。燃料电池提供 DC（直流）电压，可应用于电动机、灯或各种电气设备。有多种类型的燃料电池，每种电池使用不同的化学成分。

  • 轮胎充气 
    轮胎充气技术（中央轮胎充气）可根据变化状况快速作出调整。由于中央轮胎充气改善牵引力，因此可在劣质运输公路和坡度陡急的道路上使用。它可减少轮胎磨损。安装中央轮胎充气系统的车辆受湿地和公路环境方面的影响较少（如车扎出的沟）。

 

四、环保

监控、分析、存储和回收对环境有害的产品正日益变得复杂。此外，制造商在以环保的方式生产产品方面受到日益强劲的挑战。

Thomas 可以为环保领域所面临的挑战提供所需的：压缩机、真空泵及液体泵。

Thomas 产品典型应用包括：

  • 池塘曝气

  • 压缩机用于通过扩散器将空气注入池塘或湖内。 被注入压缩空气的池塘或湖内将增加水的含氧量，从而防止水变污浊。通风也是用于为鱼和其它水生物提供适当的生存环境，否则将缺氧。

  • 空气采样

  • 空气和灰尘采样确定建筑死角可见真菌繁殖体的数量和类型，指示室内存在严重霉菌生长区域（"放大器"）。

  • 气体分析仪

  • 这种仪器用于检查工业烟筒排放物是否有碳氢化合物进入大气。EPA 规定排放至大气中的气体中 CO2、CO、SOX、NOX 等化学物的含量。

  • 气体探测器

  • 便携式火焰离子化检测器用于测量气体样品中的碳氢化合物含量。将气体样品放在空气-氢气火焰中燃烧，可检测出碳氢化合物的存在。当使用空气-氢气混合物燃烧时导致离子化水平较高，即表示气体样品中存在碳氢化合物。离子化水平与隔膜泵抽取的样品内的碳原子数量成正比。

  • 泄漏检测

  • 在许多，必须使用保护设备监控和保护地下罐内的燃油。为防止燃油泄漏，使用双壁油罐或塑胶盖（适用于旧、单壁罐）。双壁中间形成真空，由压力检测器监控。监控设备内安装的泵时刻将压力变化和管接头及螺纹接头出现的极少量泄漏控制在某个水平。当达到超少压力时，压力检测器打开泵，当达到超大压力时，关闭泵。

  • 粒子计数器

  • 对于半导体、生物工艺学产品制造商和制药商，室内空气质量至关重要。在上述场所，"清净室"监控空气是否存在颗粒和灰尘染污物。粒子计数器与气体分析仪不同，它检测染污物的大小和数量，不识别染污物名称。将样品通过激光束，观察激光束的散射，进行检测。该应用不仅要求是流量被容易控制，而且要求气流稳定和没有脉动。

  • 冷媒回收

  • 现在，制冷剂在电动车、住宅、商业建筑内的所有空调系统中得到广泛应用，需要定期维护。 空气系统内使用的氟利昂不可再排入大气，必须在获得（政府）批准的容器内加以重新利用。

  • 污水曝气

  • 此类系统现今用于控制住宅内的废水。住宅废水进入污水处理池，注入的压缩空气促进需氧微生物的生长。需氧微生物将废水中的有机物质转化为气体和其它微生物细胞物质。除供氧外，曝气系统还使曝气区内的物质均匀混合，为微生物提供曝露在废物中的超适宜条件。污水可排入当地卫生法规准许的沟渠或新建或现有的排污场。

  • 油气回收

  • 油气回收是指车辆在加油时，将车辆油箱中释放的油气吸回的过程。油气回收可以防止臭氧的减少以及汽油中有毒污染物的释放。压缩机的起停是依靠压力传感器来检测回收罐的压力变化。油气吸入后，通过油气分离，液体被送入液体管道系统或油罐，气体通过泵回收的回收罐内。

 

五、其它应用

1、一般应用

Thomas为新兴市场提供创新型解决方案，而当今不断变革的技术，使人们对压缩机、真空泵和液体泵的需求日益增长。 

我们的产品在各种应用中得到广泛应用，如：

  • 磨皮手术

  • 磨去皮肤疤痕之手术超常见的是用于改善因事故或以前手术导致面部皮肤留下的疤痕，或消除面部细微皱纹，如嘴边的皱纹。有时它还用于去除癌症前期生长的角质。磨去皮肤疤痕之手术经常用于治疗深层的粉刺疤痕。

  • 除焊锡

  • 用于在执行维修工作时清除电路板组件上的焊料。弹簧柱塞被推进气缸和锁定。除焊工具顶部形成的真空吸走电路板熔化的焊料。

  • 抽空

  • 包括铸造时模具的抽真空，真空夹具，去除气体及蒸汽， 去除包装胶带边角料，以及罐装物真空脱泡。

  • 泡沫标记

  •   此装着用于农业生产中，在农民播种或向农作物喷农药时安装在拖拉机上。这些装置产生的泡沫球被抛掷在地面上，可使拖拉机在返回与泡沫球对齐，这样可以避免农民重复播种或过多施肥及喷农药。

  • 燃料电池

  •   燃料电池是一种化学能转化电能装置，原理是氢与氧反应生成水，在反应过程中产生电和热。燃料电池提供 DC（直流）电压，可用于电动机、灯或各种电气设备。有多种类型的燃料电池，每种电池使用不同的化学成分。

  • 浸渍

  • 金属铸件和粉状金属部件的浸渍是指堵住因孔隙导致的泄漏。液体等注入材料通过真空和压力进入部件壁内的空隙内。然后材料凝固，填满可渗透的开口，使部件耐压密封。

  • 喷墨打印

  •   喷墨打印广泛应用于工业编码、标记和标签市场。在打印过程中，每秒钟产生64,000滴墨。标记所需墨滴获得静电电荷，直接飞到介质上，而不带电的墨滴则转入隔条进行再流通。液体泵将由油墨从收集区送返储墨盒。  

  • 影印机

  • 现在超新的复印机通常采用压缩空气进行送纸、冷却上色辊、启动执行超后操作过程的滚筒等工作。

  • 照片冲洗

  • 在当今快节奏的社会，人们希望能立刻冲洗胶卷。这为一小时快照店带来勃勃商机。这些新型高科技设备可在一小时内从底片到超终印刷的胶卷冲洗工作。这些设备需要在冲洗过程中使用压缩空气冲掉底片上的纤维。

  • 等离子切割机

  •   这种设备非常适合于车身修理厂、车队、汽修厂和农场使用。它产生维弧，因此操作员可切割钢铁、铝和不锈钢。压缩空气与气体混合，形成温度很高的火焰，实现精密切割。

  • 喷雾制革法

  • 作为沙滩床的替代品，喷雾制革机正日益受到人们的欢迎，它利用系统内的压缩机，使压缩空气通过安装在喷枪之前的装有人工晒色溶液的漏斗，或将压缩空气送入喷枪本身。这种方法导致持续流动和压力释放鞣剂。

  • 牙齿美白

  • 牙齿美白过程中压缩空气和真空都用作清洁工具。强光灯去除过氧化氢凝胶混合物后，吸入泵用于清除剩下未进入牙齿的强凝胶。压缩机用于提供气流，将程序结束时出现的任何松动碎片吹除。

  • 真空晒版机

  • 从事地图绘制和规划工作的公司使用真空晒版机。储存器内安装一张"未用过的"薄膜。盖子闭合，产生真空使薄膜紧密。一旦达到所需真空，一束光通过薄膜，形成正片。正片产生的底片让您可以随意复印。

  • 真空保持

  • 这是需要按预设时间将夹具、衬垫和板固定到位的应用，以便可以执行其它工作。真空保持广泛应用于木工行业和小型金属加工厂。 

 

2、食品饮料

Thomas产品系列技术完善，满足食品饮料市场各种应用，如制氮、软饮料和啤酒配给、气体混合系统和气动设备和泵驱动用空气。

使用Thomas产品的典型OEM设备包括：

  • 咖啡机

  • 压缩机用于在过滤阶段迫使进水通过漏斗内的咖啡豆。这种煮咖啡的方法使咖啡更为香浓。

  • 饮料分配系统

  • 对于私人聚会，可选择移动式啤酒配给系统。这种机器由制冷系统和小型压缩机组成。压缩机用于使啤酒流出。有时压缩机与压力开关一同使用。

  • 冰淇淋分配系统

  • 此类系统在快餐店、购物超市和小型市场配给软冰淇淋。须使用压缩空气才能将冰淇淋推向圆筒和杯中。

  • 制氮机

  • 制氮机通过专用空气压缩机提供的压缩空气制成氮气。压缩机安装在系统开始部分，使压缩空气通过过滤器组，去除后杂质后进入制氮器。过滤后的空气再通过一个过滤器，去除氧分子和二氧化碳分子，产生高纯度氮气。
